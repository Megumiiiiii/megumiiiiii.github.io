## Delete Container + Images

```
cd ~/ar-io-node
sudo docker compose down -v
```

```bash
sudo docker image prune -a
```

```bash
sudo docker builder prune
```

## Pull

```bash
git pull
```

## Create new `.json`

- You can use same wallet as `AR_IO_WALLET=`, but it's recommended to use a new wallet

```
nano wallets/YourWalletAddress.json
```

- Example `nano wallets/a9RqasdbajdJANbhZaDbSkPaq9B8aEchPY_BgIB0Gck.json`
- Then fill it with the same Wallet Address which you use as file name.
```
{YourWalletAddress}
```

## Edit `.env`

```
nano .env
```

- Add this two line

```
RUN_OBSERVER=true
OBSERVER_WALLET=YourWalletAddress
```

## Then, UP!

```bash
docker compose up -d --build
```
