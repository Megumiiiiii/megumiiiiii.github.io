### Buat Folder & init

<Tabs>
<TabItem value="VPS">

```bash
cd ~/proc
sqd init quad-proc -t https://github.com/subsquid-quests/quad-chain-squid
cd quad-proc
```

</TabItem>
<TabItem value="CodeSpace">

```bash
cd ..
sqd init quad-proc -t https://github.com/subsquid-quests/quad-chain-squid
cd quad-proc
```

</TabItem>
</Tabs>

## Download key

- https://app.subsquid.io/quests
- Pindah ke VPS, ke folder `/proc/quad-proc/query-gateway/keys`
- ![SS](/img/squidproc.png)

## Start Docker

```bash
docker compose up -d
```

## build

```bash
npm ci
sqd build
```

```sh filename="migrate" copy
sqd migration:apply
```

### Run

```bash
sqd run .
```

Tunggu sampai di web nya jadi 100%

### Setelah selesai
 - Stop proc menggunakan CTRL+C
 - Stop Docker

 ```bash
 docker compose down -v
 ```

:::tip 
Hapus **SEMUA** jika sudah mendapat point 
:::


```bash
cd
rm -r proc
```

docker rmi subsquid/query-gateway:0.1.5
