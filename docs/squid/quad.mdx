### Create Folder & init

<Tabs>
<TabItem value="VPS">

```bash
cd ~/proc
sqd init quad-proc -t https://github.com/subsquid-quests/quad-chain-squid
cd quad-proc
```

</TabItem>
<TabItem value="CodeSpace">

```bash
cd ..
sqd init quad-proc -t https://github.com/subsquid-quests/quad-chain-squid
cd quad-proc
```

</TabItem>
</Tabs>

## Download key

- https://app.subsquid.io/quests
- Move it to VPS, into this folder `/proc/quad-proc/query-gateway/keys`
- ![SS](/img/squidproc.png)

## Start Docker

```bash
docker compose up -d
```

## build

```bash
npm ci
sqd build
```

```bash
sqd migration:apply
```

### Run

```bash
sqd run .
```

Wait after 100%

### After done
 - Stop proc use CTRL+C
 - Stop Docker

 ```bash
 docker compose down -v
 ```

:::tip 
Delete **ALL**



```bash
cd
rm -r proc
```

```bash
docker rmi subsquid/query-gateway:0.1.5
```
:::
