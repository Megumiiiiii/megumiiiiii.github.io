### Install Subsquid cli

```bash
npm install --global @subsquid/cli@latest
```

### Create Folder & init

```bash
mkdir proc
cd proc
sqd init snapshot -t https://github.com/subsquid-quests/snapshot-squid
cd snapshot
```

## Download key

<Tabs>
<TabItem value="VPS">

- https://app.subsquid.io/quests

## Start Docker

```bash
docker compose up -d
```

## build

```bash
npm ci
sqd build
```

```bash
sqd migration:apply
```

### Run

```bash
sqd run .
```

Wait after 100%

### After done
 - Stop proc use CTRL+C
 - Stop Docker

 ```bash
 docker compose down -v
 ```
