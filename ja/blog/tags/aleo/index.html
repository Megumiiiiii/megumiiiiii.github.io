Deploy Contract on Aleo without Node

2023年11月13日 · 約262分
Megumii
NPC

Install Leo Wallet​

  • Download DISINI dan install
  • Buat Wallet
  • Request faucet dengan cara kirim SMS ke nomer +18678885688
  • Kirim pesan menggunakan format
Send 50 credits to aleoxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

Mulai Deploy​

  • Setelah mendapat Aleo, pergi ke web Deployer → DISINI
  • Copy ini dan paste ke kolom Deploy
program nama_bebas.aleo;

record Token:
    owner as address.private;
    amount as u64.private;


function mint:
    input r0 as address.private;
    input r1 as u64.private;
    cast r0 r1 into r2 as Token.record;
    output r2 as Token.record;


function transfer:
    input r0 as Token.record;
    input r1 as address.private;
    input r2 as u64.private;
    sub r0.amount r2 into r3;
    cast r0.owner r3 into r4 as Token.record;
    cast r1 r2 into r5 as Token.record;
    output r4 as Token.record;
    output r5 as Token.record;
  • Ganti nama_bebas.aleo dengan nama apapun sesuka hati
  • Pastikan gunakan nama yang belum terpakai, buka Explorer lalu pada kolom Search cari nama yang akan digunakan, jika 404 berarti bisa dipakai.
  • Pastekan privkey leo wallet yang tadi dipake req faucet
  • Isi fee dengan 5
  • Matikan Private Fee
  • Deploy, done

Execute app​

  • Masukan nama app yang sudah di deploy tadi
  • Klik tombol search, maka code nya akan ter-load
  • Masukan Privkey mu ke kolom *Private Key
  • Aktifkan Execute Onchain
  • Isi fee 3
  • Lalu ke bagian mint
  • r0 isi addressmu
  • r1 isi 1000u64
  • Runnnnn
