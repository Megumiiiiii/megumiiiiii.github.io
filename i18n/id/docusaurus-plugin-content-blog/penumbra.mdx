## Install Dependencies

- any other
```bash
sudo apt-get install build-essential pkg-config libssl-dev clang git-lfs screen
```

- rustup
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

## Install pcli

```bash
git clone https://github.com/penumbra-zone/penumbra
cd penumbra && git fetch && git checkout v0.63.1
cargo build --release --bin pcli
```

### Generate/Import wallet

<Tabs>
<TabItem value="Generate Wallet">

```bash
cargo run --quiet --release --bin pcli init soft-kms generate
```

</TabItem>
<TabItem value="Import Wallet">

```bash
cargo run --quiet --release --bin pcli init soft-kms import-phrase
```

### Faucet

```bash
cargo run --quiet --release --bin pcli view address 0
```

```bash
cargo run --quiet --release --bin pcli view sync
```

```bash
cargo run --quiet --release --bin pcli view balance
```

:::info
- The minimum bid for this ceremony is 60penumbra.
- To change your bid, you can simply run the command again with a different bid amount. Note that each time you bid with the command above, this adds additional funds to all of the previous bids you've already submitted.
