## Apa yang dibutuhkan

- [Wallet](https://www.arconnect.io/download)
- `$AR`, bisa mendapatkannya dari CEX favoritmu

## Unggah ke ArDrive

- Kunjungi https://app.ardrive.io
- Pilih Login with ArConnect 
- Buat public Drive
- Kemudian unggah file Anda, bisa berupa gambar, video, file teks, apapun! (Ukuran file di bawah 500 Kb GRATIS)
- Klik file Anda dan pergi ke bagian `details`, lalu salin `Data Tx ID`
<img width="100%" height="auto" src="/img/arnsde.png" type="image/png"></img>

## Dapatkan ArNS

#### Untuk membeli ArnS, kamu perlu memiliki Token TestIO, berikut adalah cara meminta token tersebut

- Masuk ke Ar.io [Discord](https://discord.gg/c4rSSNV9v3)
    - Di channel `#arns-test`, ketik `/apply`, jawab semua pertanyaan
    - <img width="100%" height="auto" src="/img/arnsap.png" type="image/png"></img>
    - Untuk pertanyaan ini, jawab dengan Tx ID dari Langkah sebelumnya
    - <img width="100%" height="auto" src="/img/arnsas.png" type="image/png"></img>

#### Setelah menerima `Test IO`

- Kunjungi https://arns.app
- Cari nama apapun ( 8 digit++ )
- Jika tersedia, klik register now
- <img width="100%" height="auto" src="/img/arnsav.png" type="image/png"></img>
- Pilih 1 Year
- Isi `Arweave transaction ID ( Target ID )` dengan Tx ID Anda
- <img width="100%" height="auto" src="/img/arnsnt.png" type="image/png"></img>

Dan sekarang ArNS mu bisa diakses dengan `https://your-arNS.ar-io.dev`.

Contoh https://megumii.ar-io.dev

#### `ar-io.dev` dapat diganti dengan Gateways lain yang terdaftar di sini - > https://gateways.ar-io.dev
