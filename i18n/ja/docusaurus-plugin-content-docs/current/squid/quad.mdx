### Buat Folder & init

```sh filename="Node.js" copy
cd ~/proc
sqd init quad-proc -t https://github.com/subsquid-quests/quad-chain-squid
cd quad-proc
```

## Download key

- https://app.subsquid.io/quests
- Pindah ke VPS, ke folder `/proc/quad-proc/query-gateway/keys`
- ![SS](/img/squidproc.png)

## Start Docker

```sh filename="build" copy
docker compose up -d
```

## build

```sh filename="build" copy
npm ci
sqd build
```

```sh filename="migrate" copy
sqd migration:apply
```

### Run

```sh filename="run" copy
sqd run .
```

Tunggu sampai di web nya jadi 100%

### Setelah selesai
 - Stop proc menggunakan CTRL+C
 - Stop Docker

 ```sh filename="delete" copy
 docker compose down -v
 ```

:::tip 
Hapus **SEMUA** jika sudah mendapat point 
:::

```sh filename="delete" copy
rm -r proc
```
